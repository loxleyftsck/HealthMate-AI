# đź“– Project Overview

HealthMate AI adalah chatbot kesehatan berbasis AI yang membantu pengguna memperoleh edukasi kesehatan umum, rekomendasi gaya hidup sehat, serta berbagai alat pendukung seperti kalkulator BMI dan pelacak konsumsi air.

Project ini menggunakan pendekatan **UI First Development**, sehingga seluruh antarmuka dan logika frontend dikembangkan terlebih dahulu sebelum mengintegrasikan Gemini Flash API.

Project ini **bukan aplikasi diagnosis medis**, melainkan asisten edukasi kesehatan.

# âšˇ API Optimization

Untuk menghemat penggunaan Gemini Flash API:

- Tidak memanggil AI saat aplikasi dibuka.
- Quick Prompt menggunakan template lokal jika memungkinkan.
- Chat history disimpan di LocalStorage.
- Utility seperti BMI dihitung secara lokal.
- Hanya pesan yang benar-benar membutuhkan AI yang dikirim ke Gemini.
